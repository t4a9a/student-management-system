# 🎓 Student Management System

A full-stack Student Management System built using **Spring Boot** and a responsive frontend.
It allows users to manage student records with a clean UI and simple login system.

---

## 🚀 Features

* 🔐 Login system (client-side authentication)
* ➕ Add student
* 📋 View all students
* ✏️ Edit student details
* ❌ Delete student
* 🔍 Search students
* 📱 Responsive UI (mobile + desktop)
* 🎨 Modern glassmorphism design

---

## 🛠️ Tech Stack

* **Backend:** Spring Boot (Java)
* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Database:** H2 (file-based)

---

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/t4a9a/student-management-system.git
```

2. Navigate to project folder:

```bash
cd student-management-system
```

3. Run the project:

```bash
./mvnw spring-boot:run
```

4. Open in browser:

```
http://localhost:8080
```
